# Dubhe Desktop 设计记录

更新时间：2026-07-05

## 设计概念

- 概念图：[dubhe-desktop-concept.png](dubhe-desktop-concept.png)
- 实现截图：[dubhe-desktop-qa.png](dubhe-desktop-qa.png)

视觉方向：

- 中文优先的类 IDE 投资工作台。
- 深色 graphite 外壳，浅色中央分析文档。
- 左侧活动栏 + 导航树，中央多标签工作区，右侧 AI 分析师对话，底部任务日志/风控告警。
- 强调新闻分析、影响分、来源引用、策略生成、风控审批和纸面交易。

## 已实现对照

| 概念要求 | 当前实现 |
| --- | --- |
| 中文界面 | 所有主导航、按钮、状态、日志和说明均为中文。 |
| 类 IDE 布局 | 已实现活动栏、左侧栏、中央工作区、右侧对话、底部日志。 |
| 新闻分析文档 | 已展示标题、摘要、情绪、影响分、置信度、关联标的和来源引用。 |
| AI 分析师对话 | 已展示中文问答和风险解释。 |
| 风控/纸面交易 | 已接入 Core API 的风控评估与纸面订单占位链路。 |
| 底部任务/风控日志 | 已展示交互产生的实时日志，避免文字裁切。 |

## 已验证交互

使用 Playwright 在 `1440x900` 视口验证：

1. 打开 `http://127.0.0.1:5173`。
2. 点击 `分析新闻`，等待 `新闻分析完成` 日志。
3. 点击 `风控评估`，等待 `需要人工审批`。
4. 点击 `放入纸面交易`，等待 `纸面订单已接受`。
5. 检查浏览器 console/page errors：无错误。
6. 检查底部日志最后一条完整位于底部面板内。
